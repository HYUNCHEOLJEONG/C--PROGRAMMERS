이곳은 C++을 통해 알고리즘과 자료구조 문제를 풀어낸 **코드 예술의 집합소**입니다.  
단순히 정답을 맞히는 것을 넘어서, **가독성 · 효율성 · 구조적 사고**를 담아냈습니다.

- **언어**: Modern C++ (17 이상 기준)
- **문제 출처**: Programmers, 대기업 문제제등
- **목표**: 실무에서 통하는 문제해결 능력 강화
